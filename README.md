A simple responsive video library app that lets user do the following actions.

## Features

- Create Playlist
- Add Video to Playlists
- Like Videos
- Retain History of Videos
- Add to Watch Later
- Single Video Page
- Support dark mode and light mode
- Signup, Signin and Signout page.
